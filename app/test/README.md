# Test Files

This folder contains test utilities and pages for development.

## Structure

- `convex/test/` - Convex test queries and utilities
- `app/test/` - Test pages for manual testing

## Usage

**Test Generation Backend:**
Visit http://localhost:3000/test/generation

This page allows you to:

- Check configuration status
- View sample user data
- Test image generation
- Monitor job progress in real-time

**Convex Dashboard:**
Visit http://127.0.0.1:6790 when running `npx convex dev`
